Histoire des langages Informatiques ↓

  • 1843 : La création d’un algorithme par Ada Lovelace pour la machine analytique à calculer du mathématicien Charles Babbage. Sur papier. Elle a posé les bases d'aujourd'hui. => Angleterre
  • 1944 - 1945 : Konrad Zuse a inventé un langage de programmation, appelé Plankalkül, permettant de stocker des codes simples. => première vraie langue de codage à proprement parler. Allemagne
  • 1949 : Invention de « code court »(«shortcode») par John McCauley . Le premier langue de haut niveau.
  • 1957 : Invention du langage « fortran » par John Backus. Fortran = Formule Transfert => langue crée pour les calculs scientifiques et statistiques de haut niveau.
  • 1964 : Création du langage « Basic » qui est un code d’instruction universel pour débutants crée au Dartmouth Collège par Bill Gates et Paul Allen.
  • 1972 : Développement du langage « C » par Dennis Ritchie au Laboratoire Bell Telephone (plusieurs laboratoires fondés en 1925 au New Jersey ayant déposés +30 000 brevets).
  • 1974 : L’année de la création du langage « SQL » (Structured Query Language) par Donald Chamberlin et Raymond Boyce.
  • 1983 : Création d’ « Objectif-C » par Brad Cox et Tom Love, est le langage de programmation pour écrire les logiciels et les système d’exploitation d’Apple. USA
  • 1991 : Création de Python par Guido Van Rossum en tant qu’outil d’usage général. Deuxième langage de programmation la plus utilisé dans le monde. Pays Bas
  • 1993 : Création du World Wide Web et du langage HTML par la world wide web consortium, a été conçu pour représenter les pages web et créer des liens hypertextes (connectant les pages entres elles).
  • 1995 : Développement de « Java » par James Gosling. Langage universel de haut niveau.
  • 1995 : Création du codage « JavaScript » par Brendan Eich. JavaScript est utilisé pour le développement du world wide web, les documents pdf, les navigateur web ext…
  • 2000 : Création du langage C# par Microsoft dans le but de combiner le langage C++ avec le langage « Visual Basic » un langage pas très connu mais assez simple.
  • 2009 : Année d’officialisation de « GO » développé par Google pour résoudre un problème lié aux grands systèmes logiciels, grâce à une simplicité hors norme.
  • 2014 : Apple créé le codage « Swift » pour remplacer C, C++ et Objective-C. Swift a une simplicité digne de ce nom, comparable à GO.
  • 2017 : Création de QCL (Quantum Computing Language) par Bernhard Ömer. Ressemblant beaucoup au langage C au niveau de la syntaxe, il y diffère par l'ajout de nombreuses opérations permettant d'explorer les concepts de programmation pour les ordinateurs quantiques.
